# 味溯新东方

一款便捷的菜谱管理应用，帮助您轻松管理、编辑和烹饪美味佳肴。

## 功能特性

### 核心功能
- 菜谱浏览 - 查看完整的菜谱列表，支持搜索和分类筛选
- 菜谱详情 - 查看菜谱的食材、烹饪步骤和小贴士，支持收藏
- 在线编辑 - 直接在应用内编辑和完善菜谱内容
- 导入导出 - 支持将菜谱导入和导出为纯文本
- 标签管理 - 为菜谱添加标签，方便分类和查找
- 备料检查 - 烹饪前的备料清单检查，食材一键加入采购清单
- 烹饪指导 - 分步烹饪指导，支持断点续做
- 食材库存 - 管理家中现有食材，支持数量/单位/保质期编辑和过期提醒
- 采购清单 - 按菜谱生成的购物列表，支持一键移入库存
- 我的收藏 - 收藏喜爱的菜谱，点击跳转详情，支持反复切换
- 烹饪笔记 - 自由便签本记录心得，配方区可查看编辑过的菜谱
- 计划菜单 - 规划一周菜单
- 用户反馈 - 一键发送反馈邮件给开发者
- 最近打开排序 - 打开过的菜谱自动排序到列表最前

### 技术特点
- 跨平台 (Android/iOS/Web)
- React Native + Expo 开发
- 本地存储，数据安全
- 美观的用户界面
- 标签驱动的分类筛选体系

## 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn
- Android Studio (用于Android开发)
- Xcode (用于iOS开发，仅Mac)

### 安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 运行Android版本
npm run android

# 运行iOS版本
npm run ios

# 运行Web版本
npm run web
```

## 菜谱格式说明

### 导出格式示例
```
番茄炒蛋

📝 简介
家常菜经典款，酸甜可口

🗺️ 主体流程
备料→炒蛋→炒番茄→混合调味

⏱️ 基本信息
准备时间	10分钟
烹饪时间	5分钟
份量	2人份
难度	 简单
烹饪技法	炒
菜肴味型	酸甜

食材清单：

主料
鸡蛋	3个
番茄	2个

辅料
葱花	适量

调料
盐	适量
糖	1	小勺
胡椒粉	1	勺尖
料酒	几滴
番茄酱	2大勺
生抽		1小勺

📋 备料步骤
第 1 步：番茄处理 番茄洗净，在顶部划十字刀；放入开水中烫 30 秒后捞出，轻松剥去外皮，切滚刀块或月牙形备用。 耗时：2 分钟 💡 小贴士：烫过更容易去皮，口感也更细腻
第 2 步：鸡蛋处理 鸡蛋 3 枚打入碗中，加少许盐、胡椒粉、几滴料酒，搅匀至起泡备用。 耗时：1 分钟 💡 小贴士：蛋液里加半勺淀粉，炒出来更嫩滑
第 3 步：配料准备 葱切葱花，姜切末，番茄酱、盐、白糖、生抽备好放旁边。 耗时：1 分钟 💡 小贴士：调料提前备好，炒菜时不慌乱

🍳 制作步骤
第 1 步：炒鸡蛋：热锅凉油，油温升至七成热，倒入蛋液；待底部凝固后轻推翻炒，炒至八分熟、金黄蓬松即可盛出备用。 耗时：1 分钟 💡 小贴士：油温七成热（约 180°C）下锅，不要全熟，留待余温即可凝固，避免蛋炒老
第 2 步：煸炒番茄：锅中留底油，下葱姜爆香，倒入番茄翻炒；加少许盐逼出汤汁，大火翻炒至番茄变软出沙。 耗时：2 分钟 💡 小贴士：番茄要炒出红汤，味道才浓郁；偏爱酸可加1~2勺番茄酱提味
第 3 步：合炒调味：鸡蛋回锅，加盐 1 小勺、白糖半小勺、少许生抽，大火翻炒均匀，让蛋充分吸收番茄汁。 耗时：1 分钟 💡 小贴士：糖的用量依口味调整，北方口味可省略
第 4 步：爱吃无汤汁，转大火稍收汁，撒葱花点缀，出锅装盘即可上桌。 耗时：30 秒 💡 小贴士：喜欢汤汁多的不收干，直接盛出更下饭

🏷️ 标签
家常菜、快手菜、下饭菜
```

### 格式说明
- **第一行**：菜谱名称
- **📝 简介**：菜谱描述（可选）
- **🗺️ 主体流程**：整体烹饪流程描述
- **⏱️ 基本信息**：准备时间、烹饪时间、份量、难度、烹饪技法、菜肴味型
- **食材清单**：分类食材（主料/辅料/调料），名称和用量使用tab分隔
- **📋 备料步骤**：备料操作指南，可添加耗时和小贴士
- **🍳 制作步骤**：烹饪操作步骤，可添加耗时和小贴士
- **🏷️ 标签**：菜谱标签，使用顿号或逗号分隔
