# 炒菜助手 🍳

一款便捷的菜谱管理应用，帮助您轻松管理、编辑和烹饪美味佳肴。

## 功能特性

### 核心功能
- 📖 **菜谱浏览** - 查看完整的菜谱列表，支持搜索和分类筛选
- 🍽️ **详细信息** - 查看菜谱的食材、烹饪步骤和小贴士
- 📝 **在线编辑** - 直接在应用内编辑和完善菜谱内容
- 💾 **导出导入** - 支持将菜谱导出为纯文本，以及从文本导入
- 🏷️ **标签管理** - 为菜谱添加标签，方便分类和查找
- 📋 **备料检查** - 烹饪前的备料清单检查
- 🔥 **烹饪指导** - 分步烹饪指导，确保菜品完美呈现

### 技术特点
- 📱 跨平台 (Android/iOS/Web)
- 🚀 React Native + Expo 开发
- 💾 本地存储，数据安全
- 🎨 美观的用户界面

## 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn
- Android Studio (用于Android开发)
- Xcode (用于iOS开发，仅Mac)

### 安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 运行Android版本
npm run android

# 运行iOS版本
npm run ios

# 运行Web版本
npm run web
```

## 菜谱格式说明

### 导出格式示例
```
番茄炒蛋

📝 简介
家常菜经典款，酸甜可口

⏱️ 基本信息
准备时间	10分钟
烹饪时间	5分钟
份量	2人份
难度	简单
分类	家常菜

🥦 食材清单
鸡蛋	3个
番茄	2个
盐	适量

📋 备料步骤
番茄洗净，在顶部划十字刀
将番茄放入开水中烫30秒 💡 小贴士：更容易去皮

🍳 炒菜步骤
热锅凉油，倒入蛋液 耗时：1分钟 💡 小贴士：油温七成热

🏷️ 标签
家常菜、快手菜、下饭菜
```

### 格式说明
- **第一行**：菜谱名称
- **📝 简介**：菜谱描述（可选）
- **⏱️ 基本信息**：准备时间、烹饪时间、份量、难度、分类
- **🥦 食材清单**：食材名称和用量，使用tab分隔
- **📋 备料步骤**：备料操作指南，可添加小贴士
- **🍳 炒菜步骤**：烹饪操作步骤，可添加耗时和小贴士
- **🏷️ 标签**：菜谱标签，使用顿号或逗号分隔

## 项目结构

```
CookHelper/
├── assets/             # 资源文件
│   ├── icon.png       # 应用图标
│   └── recipes.txt    # 初始菜谱数据
├── src/                # 源代码
│   ├── screens/       # 页面组件
│   ├── context.tsx    # 状态管理
│   ├── navigation.tsx # 路由配置
│   ├── parseRecipe.ts # 菜谱解析
│   ├── storage.ts     # 数据存储
│   └── types.ts       # 类型定义
├── android/           # Android平台代码
├── App.tsx            # 应用入口
└── app.json          # 应用配置
```

## 主要技术栈

- **React Native 0.81** - 跨平台移动应用框架
- **Expo 54** - 开发工具和平台
- **React Navigation** - 导航库
- **AsyncStorage** - 本地数据存储
- **TypeScript** - 类型安全的JavaScript
